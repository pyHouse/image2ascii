# ✍ 图片转字符画 - Python3

- 🔌 Install
```
pip install argparse
pip install pillow
```

- 🖥 Usage:
```
python ascii.py <input_file> <out_file> [--width WIDTH] [--height HEIGHT]
```

- 📖 Options:
```
-h --help       显示帮助
-width          图片压缩的宽度
-height         图片亚索的高度
```

- ⌨ Example:
```
python ascii.py test.jpg out_file
python ascii.py test.jpg out_file --width 80 --height 80
```
